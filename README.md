# Aaron's Dotfiles

Dotfiles for Niri + Noctalia on PikaOS (Debian-based).

## Install

Clone the repository and run the install script:

```bash
git clone https://github.com/theblack-don/seabee-dots.git
cd seabee-dots
bash install.sh
```

## What it does

- Installs: `niri`, `noctalia-shell`, `xwayland-satellite`, `hx`, `fastfetch`, `fish`, `qt5ct`, `qt6ct`, `qml6-module-qt-labs-folderlistmodel`, `qt6-wayland`, `kitty`, `nemo`
- Copies all configs from `dotfiles/` into `~/.config/`
- Copies wallpapers from `wallpapers/` into `~/Pictures/Wallpapers/`
- Sets Noctalia wallpaper directory and default wallpaper (`37.png`)
